# libzen 开发日志

## 2026-03-21

### 第三阶段（续）+ 第四阶段：error、threading 模块

#### 1. error/system_error.h
- error_category 抽象基类
- errc 枚举（30+ 常用错误码）
- generic_category_impl 单例
- error_code（值类型：int 值 + 错误类别指针）
- error_condition（逻辑层错误条件）
- make_error_code() / ok() 工厂函数

#### 2. error/error_context.h
- error_context（error_code + 文件名 + 行号 + 函数名 + 消息）
- 固定大小 char 数组存储，零堆分配
- print()/format() 输出方法
- ZEN_MAKE_ERROR / ZEN_OK / ZEN_RETURN_IF_ERROR 宏

#### 3. error/exception_chain.h
- error_chain：最多 MAX_DEPTH=16 层的错误传播链
- top() / root_cause() / print_all()
- result<T>：Rust 风格 Ok/Err 返回类型
- result<void> 特化

#### 4. threading/sync/mutex.h
- mutex（非递归，POSIX pthread / Win32 CRITICAL_SECTION）
- recursive_mutex（PTHREAD_MUTEX_RECURSIVE）
- timed_mutex（带超时，try_lock_for 毫秒）

#### 5. threading/sync/lock_guard.h
- lock_guard（简单 RAII）
- unique_lock（灵活：defer_lock/adopt_lock/try_to_lock，支持移动）
- scoped_lock（同时锁多个 mutex，按指针地址排序避免死锁）
- defer_lock / adopt_lock / try_to_lock 标签

#### 6. threading/sync/spinlock.h
- spinlock（TTAS 自旋锁，CAS + cpu_relax + memory_fence）
- ticket_lock（公平自旋锁，FIFO 排队）
- 内嵌汇编：x86 pause、ARM yield、mfence

#### 7. threading/sync/condition_variable.h
- condition_variable（POSIX pthread_cond / Win32 CONDITION_VARIABLE）
- wait(ul) / wait(ul, pred) / wait_for(ul, ms) / wait_for(ul, ms, pred)
- notify_one() / notify_all()

#### 8. threading/sync/rwlock.h
- rwlock（POSIX pthread_rwlock / Win32 SRWLock）
- shared_mutex 别名
- shared_lock<M>（读锁 RAII）
- write_lock<M>（写锁 RAII）

#### 9. threading/thread/thread.h
- thread 类（RAII，POSIX pthread / Win32 CreateThread）
- 类型擦除 thread_task 系列（支持无参/1参/2参 callable）
- join() / detach() / joinable() / get_id()
- hardware_concurrency()
- thread_id 类型

#### 10. threading/thread/this_thread.h
- get_id() / sleep_for(ms) / sleep_for_us(us) / yield()
- monotonic_ms() / monotonic_us()（单调时钟）
